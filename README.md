## 서비스 소개

졸업생, 재학생, 지원자가 서로 정보를 공유할 수 있는 커뮤니티로, 대학원을 지원하는 사용자에게 도움이 될 수 있는 기능을 제공하는 IT 플랫폼 입니다.

## 🖥️ 개발 환경

- Xcode: 15+
- Swift: 5.9+
- 배포타겟: iOS 17+
- 의존성 관리: CocoaPods
- 작업 일정관리: Notion

<br>

## 🔑 핵심 기술 

### 🗃️ 프레임워크
- UI: UIKit
- 암호화: Security
- 카카오로그인: KakaoSDK

### 🗂️ 외부 의존성
- UI: Then, SnapKit, DropDown, Tabman
- 네트워크 통신: Alamofire
- 비동기 이벤트: RxSwift
- 단방향 데이터 흐름: ReactorKit
